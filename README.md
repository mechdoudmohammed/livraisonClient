# Project for Location des Voitures
# Installation

First clone this repository, install the dependencies, and setup your .env file:

1- git clone https://github.com/mechdoudmohammed/LocationVoiture.git<br>
2- composer install<br>
3- npm install<br>
4- Copy .env.example file to .env on the root folder. You can type copy .env.example .env if using command prompt Windows or cp .env.example .env<br>
5- Open your .env file and change the database name (DB_DATABASE) to whatever you have, username (DB_USERNAME) and password (DB_PASSWORD) field correspond to your configuration.<br>
6- php artisan key:generate<br>
7- php artisan migrate<br>
8- php artisan serve<br>
9- Go to http://localhost:8000/<br>


Then add some data to database for testing:<br>
